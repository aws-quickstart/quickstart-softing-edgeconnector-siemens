(`QSS3BucketName`)|`aws-quickstart`|Name of the S3 bucket for your copy of the Quick Start assets. Keep the default name unless you are customizing the template. This name can include numbers, lowercase letters, uppercase letters, and hyphens, but do not start or end with a hyphen (-). (`QSS3BucketRegion`)|`eu-central-1`|AWS Region where the Quick Start S3 bucket (QSS3BucketName) is hosted. Keep the default Region unless you are customizing the template. (`QSS3KeyPrefix`)|`quickstart-softing-edgeconnector-siemens/`|S3 key prefix that is used to simulate a directory for your copy of the Quick Start assets. This prefix can include numbers, lowercase letters, uppercase letters, hyphens (-), and forward slashes (/). End with a forward slash. See https://docs.aws.amazon.com/AmazonS3/latest/dev/UsingMetadata.html and https://aws-quickstart.github.io/option1.html.
